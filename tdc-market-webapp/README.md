# TDC Market - Web App

Türkiye'nin Tasarım & Figür Pazarı - Next.js 14 Web Application

## Features

- Next.js 14 with App Router
- Tailwind CSS with TDC Design System
- TypeScript
- Responsive Design
- SEO Optimized
- Performance Optimized

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Deploy

This app is optimized for Vercel deployment.
