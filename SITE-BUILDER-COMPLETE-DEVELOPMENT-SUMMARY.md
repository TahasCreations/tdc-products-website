### Typography Manager
```
1. Properties panel'de Typography section'ı aç
2. Font family seç
3. Size preset'lerden seç
4. Weight ve spacing ayarla
5. Live preview'da gör
```

```
1. Media section'ı aç
2. Dosyaları sürükle & bırak
```

```
1. Form component'i ekle
```
